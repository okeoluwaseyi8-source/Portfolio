# Pizza Sales & Order Analysis (SQL)

## Objective
Analyze pizza sales and order data to compute key KPIs such as revenue,
order volume, average order value, and average items per order.

## Tools Used
- SQL (SQL - Server)

## Key KPIs
- Total Revenue
- Total Orders
- Average Order Value
- Average Pizzas per Order
