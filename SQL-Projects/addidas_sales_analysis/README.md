# Adidas Sales Analysis (SQL)

## Project Overview
This project performs a comprehensive SQL analysis of Adidas sales data for 2021.
The analysis answers key business questions related to revenue, profitability,
product performance, pricing, and sales trends.

## Dataset
- **Source:** Adidas sales dataset
- **Table:** `Data Sales Addidas`
- **Time period:** 2021

## Tools Used
- SQL Server

## Business Questions Answered
This analysis answers questions such as:
- Which products and retailers generate the highest revenue?
- What are the best- and worst-performing products by profit and margin?
- How do sales and units sold vary by month?
- Which product categories drive operating profit?
- How pricing and operating margin relate to performance

## Analysis Approach

- Time-based filtering by month, quarter, and year
- Product, retailer, and region-level analysis
