# Product Sales Dashboard (Power BI)

## Overview
This Power BI dashboard analyzes product sales performance for a sound
equipment company, with the view filtered to **2023**.
The dashboard focuses on revenue performance, profitability, discount impact,
and customer segmentation.

## Data Preparation
The dataset used in this dashboard was prepared using SQL:
- SQL Project: ../SQL-Projects/product-analysis

## Dashboard Scope (2023)
All visuals and KPIs reflect sales performance for the year **2023**, with the
option to extend the analysis to other years using filters.

## Key Insights (2023)
- Revenue performance shows clear monthly variation across 2023
- Discounted sales are primarily driven by **high** and **medium** discount bands
- Government and Enterprise customer segments generate the highest revenue
- Sales performance differs significantly across countries
